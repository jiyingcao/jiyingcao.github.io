〇、Node.js安装之坑

操作系统是Ubuntu 19.10。一开始我通过NodeSource安装了Node.js，执行npm install -g hexo-cli之后提示权限不足：

/usr/lib/node_modules目录没有写权限，这也不奇怪；Hexo的文档强烈不建议用sudo或是root用户权限解决问题，给出的方案是用版本管理软件（Node version manager）安装Node.js。我选的是nvm：

这样Node.js的安装位置是~/.nvm/versions/node/v12.16.0/，成功避免了EACCES权限问题。

npm下载依赖包很慢，建议配置国内镜像。可以安装nrm，nrm use taobao切换到淘宝NPM镜像。感谢@夜雨随风

一、GitHub Pages之坑
